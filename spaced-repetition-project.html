Spaced Repetition Project

Development Stack: JavaScript, React, Node, Express, PostgreSQL, custom built API, Vercel, Heroku, linked list data structure

Description: This spaced-repetition application is a RESTful API powered flashcard learning application bootstrapped with React and custom JavaScript. On the backend, this application utilizes PostgreSQL relational databases and Express to make HTTP requests to the server. The spaced repetition nature of this project uses a singly linked list data structure. Deployment for the client is handled using Vercel while deployment for the backend is handled using Heroku.
